# Hack4Freedom Resources

This repository contains open, community-curated learning resources for Hack4Freedom and the broader Bitcoin and freedom technology ecosystem.

It supports female builders across the Global South who are learning to build, contribute, and work in open-source projects using **Bitcoin, Lightning, Nostr, and ecash**.

This repository is **open-source and open to contributions**.

## Why This Repo Exists

Learning Bitcoin and freedom tech can feel fragmented, overwhelming, and gatekept, especially outside traditional tech hubs.

Hack4Freedom Resources exists to:
- Lower the barrier to entry for new builders
- Provide practical, builder-focused learning material
- Curate high-quality resources in one place
- Make Bitcoin education more accessible to everyone in the world
- Support learning before, during, and after Hack4Freedom hackathons

---

## Who This Is For

This repository is for:
- Hack4Freedom participants and alumni
- Software developers entering Bitcoin and freedom tech
- Open-source contributors and maintainers
- Educators and mentors
- Anyone learning to build on Bitcoin in the Global South

No prior Bitcoin experience is required.

## Learning Philosophy

This repo follows a few simple principles:

- **Education before acceleration**  
  We value understanding over speed.

- **Practice over theory**  
  Resources should help people build things.

- **Open-source first**  
  Public docs, public repos, public learning.

- **Global South-friendly**  
  Accessible, realistic, and respectful of local constraints.

## How to Contribute

1. Fork this repository  
2. Add your resource to the appropriate folder  
3. Follow the format guidelines below  
4. Open a pull request describing why the resource is useful  

All contributions are reviewed by the Hack4Freedom organizing team (Evento).
