# AI

This folder contains resources related to using artificial intelligence and machine learning in freedom technology projects.

Resources here focus on **practical, responsible uses of AI** that support:
- Financial access and inclusion
- Developer productivity
- User education and onboarding
- Data analysis and decision support

Topics may include:
- Applying AI to Bitcoin and Lightning applications
- Using AI for fraud detection, UX improvements, or operational tooling
- Open-source AI tools and models
- Privacy-preserving and local-first AI approaches
- Ethical considerations when deploying AI in sensitive contexts

This folder prioritizes **open-source, privacy-respecting, and transparent AI tools** over proprietary or opaque systems.
